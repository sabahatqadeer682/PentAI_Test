# PentAi
                                                                                                         
  A landing page project built from a Figma design.         
                                                                                                       
  **Live:** https://pent-ai-test.vercel.app                                                              
  
  -----                                                                                                 
                                                            
  ## Overview                                                                                          

  This is the marketing landing page for PentAi. I built it from the Figma file to match the design as   
  closely as possible, with smooth hover interactions, custom gradients, and full responsive support for desktop and mobile.                                                                                    
                                                            
  The page includes a hero, supporters strip, dashboard preview, stats, security section, features,    
  testimonials, and CTA.

  -----

  ## Stack

  - Next.js                                                                                              
  - Tailwind CSS
